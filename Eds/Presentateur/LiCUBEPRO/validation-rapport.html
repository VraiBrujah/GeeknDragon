🚀 Comment Utiliser le Système

1. Pour Changer l'Emplacement des Images

Méthode Simple :
Modifier la ligne imagePath: 'image/Produit/Li-CUBE PRO/' dans js/image-config.js
Résultat : Toutes les images sont automatiquement mises à jour !

2. Pour Ajouter une Nouvelle Image

1. Ajouter le fichier image dans le dossier
2. Ajouter l'entrée dans availableImages dans js/image-config.js
3. Utiliser data-licube-image="nouveau-fichier.png" dans le HTML

3. Pour Tester le Système

Ouvrir test-image-system.html dans un navigateur pour :
• Valider le fonctionnement
• Tester les changements d'emplacement
• Voir le statut de toutes les images
• Utiliser la console de debug
